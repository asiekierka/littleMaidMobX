座れるクッションMOD Zabuton 1.6.2 Rev2

ちゃんと座れる座布団を追加するMOD
ソース付き、いっぱい家具を増やそうぜ？
あと、ディスペーンサーで射出できるように成りました。
座布団は投げすてるモノ。
さらに、水没させれば乗っているMOBを解放するように成りました。
でも座布団は流されません。


利用条件
	・テクスチャーを作ってくれたおこた氏に感謝の祈りを捧げてください。
	・動画等での使用、改造、転載すきにしてもよいのよ？
	・ただし、商用利用は除く。
	・あと、いかなる意味でも作者は責任をとりませぬ。


使い方
	要Modloader。
	要MMMLib。
	%appdata%/.mincraft/versions/1.6.2ML/modsにZIPのまま放りこめば動くはずです。
	%appdata%/.mincraft/config/mod_zabuton.cfgができるので設定はそちらで。


効能
	・同色の羊毛２コを横に並べ、その上に糸を一つ配置するレシピにも出来ます。
	・羊毛の色で色違いの座布団が作れます。
	・色毎にテクスチャが別になっています。
	・ディスペンサーから発射できます。
	・燃えます。
	・水没でMOBを開放します。


レシピ
	S
	WW

		S : String(糸)
		W : Wool(羊毛)


注意
	・バックアップはこまめに取りましょう。
	・ZabutonのshiftedIndexは22202です、アイテムIDは21946になります。
	・アイテム追加系MODなのでバージョンアップ前にはこのMODで作成したアイテムは消しておきましょう。
	・テクスチャの取り扱いが変わっています、差し替えていたヒトはご注意下さい。
	・レシピが変わっています。
